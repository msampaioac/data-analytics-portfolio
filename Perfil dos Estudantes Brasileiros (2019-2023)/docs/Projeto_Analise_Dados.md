# Projeto de Análise de Dados

Este projeto integra dados do Censo Escolar, Censo da Educação Superior
e Diretório de Municípios utilizando SQL Analítico em BigQuery para
construir uma base consolidada sobre o perfil dos estudantes brasileiros
entre 2019 e 2023.

Inclui integração de tabelas, indicadores demográficos, geográficos,
institucionais e de infraestrutura escolar, possibilitando uso em
Business Intelligence e Analytics.

# Perfil dos Estudantes Brasileiros (2019–2023)

Projeto de Análise de Dados desenvolvido com SQL no Google BigQuery utilizando dados públicos do INEP disponíveis na Base dos Dados.

## Objetivo
Construir uma base analítica consolidada sobre o perfil dos estudantes brasileiros dos ensinos Fundamental, Médio e Superior.

## Tecnologias
- SQL
- Google BigQuery
- Base dos Dados
- Business Intelligence
- Git/GitHub
- Inteligência Artificial (apoio ao desenvolvimento)

## Estrutura
- sql/: consultas SQL
- docs/: documentação e dicionário de dados
- prompts/: prompts utilizados
- datasets/: amostras de dados
- powerbi/: dashboards
